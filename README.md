# Claude CI/CD Demo

This repository demonstrates a minimal GitHub Actions workflow that can invoke Claude for PR and issue automation.

## Contents
- A sample GitHub Actions workflow for Claude-based review automation
- One intentionally insecure PHP file for demonstration purposes

## Setup
1. Add the `ANTHROPIC_API_KEY` secret to the repository.
2. Optionally add `CLAUDE_GITHUB_TOKEN` if you want to customize permissions.
3. Push changes to trigger the workflow.

> The insecure file is included only for demonstration and testing scenarios.
